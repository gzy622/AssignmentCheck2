# 作业极速登记 Pro

一个单文件（`index.html`）的课堂作业登记工具，面向教师在课上快速标记作业提交状态和分数。

## 功能概览

- **网格化点名登记**：按学号/姓名展示学生卡片，点击即可切换“已完成”状态。
- **分数登记模式**：开启“打分”后可直接录入学生分数，并可联动提交状态。
- **多作业项目管理**：支持新增、切换、重命名、删除作业项目。
- **名单编辑**：支持批量编辑学生名单（每行一条：`学号 空格 姓名`）。
- **统计视图**：按作业项目查看班级平均完成率、学生完成率和提交明细。
- **本地持久化**：数据保存在浏览器 `localStorage`，刷新页面后仍保留。
- **备份导入导出**：支持导出为 `backup.json`，也可导入恢复。
- **动画开关**：可切换界面动画效果，适配低性能设备。

## 快速开始

本项目无需安装依赖，直接运行静态页面即可。

### 方式一：直接打开

1. 在文件管理器中双击 `index.html`。
2. 使用浏览器打开后即可开始登记。

### 方式二：本地静态服务（推荐）

在项目目录执行：

```bash
python3 -m http.server 8000
```

然后访问：`http://localhost:8000`

## 使用说明

### 顶部操作

- **作业下拉框**：切换当前作业项目。
- **打分**：开启后，点击学生卡片可录入分数。
- **姓名**：在“学号视图 / 姓名视图”间切换。
- **右上角菜单**：进入统计、作业管理、名单编辑、导入导出、动画开关。

### 学生名单格式

在“编辑学生名单”中，每行一名学生，推荐格式：

```text
01 张三
02 李四
03 王五
```

若一行只有学号（无空格和姓名），系统也可识别。

### 数据存储与备份

- 页面数据默认存储在当前浏览器的 `localStorage`。
- 更换浏览器、清理缓存或使用无痕模式，可能导致本地数据丢失。
- 建议定期通过“导出备份”保存 `backup.json`。
