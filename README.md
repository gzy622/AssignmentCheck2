# 作业极速登记 Pro

一个单文件（`index.html`）的课堂作业登记工具，面向教师在课上快速标记作业提交状态和分数。页面无需构建，直接在浏览器中运行。

## 功能概览

- **网格化点名登记**：按学号/姓名展示学生卡片，点击即可切换“已完成”状态。
- **分数登记模式**：开启“打分”后可逐个录入学生分数，支持快速设为 `0`、`100`、`+1`、`-1`，并联动提交状态。
- **多任务管理**：支持新增、切换、重命名、删除任务，并可维护任务科目标签。
- **名单编辑**：支持批量编辑学生名单（每行一条：`学号 空格 姓名`），可为个别学生添加排除标记。
- **统计视图**：支持按多个任务汇总查看班级平均完成率、学生完成率和逐项提交明细。
- **本地持久化**：数据保存在浏览器 `localStorage`，刷新页面后仍保留。
- **备份导入导出**：支持导出为 `backup.json`，导入时会校验数据结构后再覆盖现有数据。
- **动画开关**：可切换界面动画效果，适配低性能设备。
- **移动端适配**：网格会根据视口自动调整尺寸，弹窗针对移动端输入做了专门处理。

## 快速开始

本项目无需安装依赖，直接运行静态页面即可。

### 方式一：直接打开

1. 在文件管理器中双击 `index.html`。
2. 使用浏览器打开后即可开始登记。

### 方式二：本地静态服务（推荐）

在项目目录执行：

```bash
python3 -m http.server 8000
```

然后访问：`http://localhost:8000`

## 使用说明

### 顶部操作

- **任务下拉框**：切换当前任务。
- **打分**：开启后，点击学生卡片可录入分数。
- **姓名**：在“学号视图 / 姓名视图”间切换。
- **右上角菜单**：进入新建任务、统计、任务管理、名单编辑、导入导出、动画开关。

### 任务与科目规则

- 新建任务时默认按当天日期生成任务名，并默认科目为“英语”。
- 任务管理中可修改任务名称和科目，科目支持“英语 / 语文 / 数学 / 物理 / 化学 / 其他”预设，也可手动输入。
- 当任务科目为“英语”时，名单中带 `#非英语`、`#非英`、`#no-en`、`#noeng`、`#not-en` 标记的学生会自动排除，不参与登记和统计。

### 学生名单格式

在“编辑学生名单”中，每行一名学生，推荐格式：

```text
01 张三
02 李四
03 王五
```

若一行只有学号（无空格和姓名），系统也可识别。

排除英语任务的示例：

```text
01 张三
02 李四 #非英语
03 王五 #no-en
```

### 数据存储与备份

- 页面数据默认存储在当前浏览器的 `localStorage`。
- 更换浏览器、清理缓存或使用无痕模式，可能导致本地数据丢失。
- 建议定期通过“导出备份”保存 `backup.json`。
- 备份文件包含学生名单 `list` 和任务数据 `data`；导入时会检查任务 ID、任务名称、记录结构等基础合法性。

### 统计与登记说明

- 主页顶部进度条显示当前任务的完成进度。
- 统计页可同时选中多个任务，按有效参与学生计算完成率。
- 英语任务中的排除学生不会计入总人数，也不会影响统计结果。
- 打分模式下，录入分数会自动将该学生标记为已完成；也可以单独切换完成状态。
