1. 完成代码修改后，按照IDE系统时间更新项目版本号。

2. DOM 动画时序问题防范：
   - 避免在 CSS 过渡/动画期间依赖 `getBoundingClientRect()` 获取尺寸
   - 解决方案：使用 CSS 变量预设尺寸，或等待动画完成后再测量
   - 示例：滑动面板类组件应在显示前预设高度变量，而非动画中测量

3. 键盘交互规范：
   - 模态/面板组件必须支持 Escape 键关闭
   - 表单/输入组件应支持 Enter 键确认提交
   - 通过 `keydown` 事件监听实现，注意调用 `preventDefault()` 阻止默认行为

4. 输入框设计原则：
   - 如需支持非数字输入，避免使用 `inputmode="none"` 或 `type="number"`
   - 使用 `readonly` 控制编辑状态时，需同步处理 `oninput` 事件确保数据一致性
